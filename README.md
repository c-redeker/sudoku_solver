# Sudoku Solver (in progress)
This repo contains a solver for sudokus (not finished yet).

## Dependencies
[googletest](https://github.com/google/googletest)

## Installation
```bash
mkdir _build
cd _build
cmake ..
make
```

## Usage
from build directory _build run
```bash
./sudoku_solver_main sudokus.txt
```
